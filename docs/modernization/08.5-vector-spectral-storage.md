# Stage 8.5 — Vector spectral storage

This increment replaces `onedtotwod_vrtdiv` and `twodtooned_vrtdiv` with
maintained free-form Fortran in `src/modern/spharm_vector_spectral_storage.f90`.

For total degree `n >= 1`, the compact triangular mapping keeps the historical
signs, `0.5` scaling and `sqrt(n(n + 1))` degree factor. The degree-zero
vorticity/divergence coefficient is defined as zero in both directions because
it does not represent a vector-harmonic mode.

The historical F77 files remain as provenance only and are excluded from the
Meson and fixed-to-free conversion manifests.

Tests cover factor/sign mapping, multiple fields, round trips, zero-mode
normalization and invariance of `Spharmt.getuv` to global vector coefficients.
