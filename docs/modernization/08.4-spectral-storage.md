# Stage 8.4 — Scalar spectral storage

This increment replaces the compiled `onedtotwod` and `twodtooned` adapters
with `src/modern/spharm_spectral_storage.f90`.

The external F2PY names and argument order are unchanged. The implementation
preserves the compact triangular `(m, n)` ordering and the historical scale:
expansion divides each complex coefficient by `0.5`; compaction multiplies the
real/imaginary pair by `0.5`.

The F77 files `src/onedtotwod.f` and `src/twodtooned.f` remain in the source
tree as provenance only. They are removed from the Meson and fixed-to-free
conversion manifests.

Only the scalar pair is changed here. The vorticity/divergence conversion pair
remains on the legacy path because its degree-zero handling needs an isolated
numerical characterization.

`tests/test_spectral_storage.py` verifies the compact index mapping, the scale
factor, multiple fields, and exact compact-to-expanded-to-compact round trips.
All normal Stage 8 acceptance gates remain required, including
`legacy-contract-v1` and installation tests for wheel, sdist and editable
builds.
